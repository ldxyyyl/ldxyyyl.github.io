应用Hexo+Github搭建博客

  • 2024-01-24

1、安装Git[地址: Git - Downloading Package (git-scm.com)]
2、安装node.js [地址: Node.js (nodejs.org)]
3、安装Hexo
（1）命令: npm install -g cnpm –registry=https://registry.npm.taobao.org（使用淘宝镜像安装 cnpm）
（2）命令：cnpm install -g hexo-cli（安装Hexo）
（3）命令：hexo init（初始化文件夹）
（4）命令：npm install（安装必备的组件）
这时文件夹下有如下内容
node_modules: 依赖包
public：存放生成的页面
scaffolds：生成文章的一些模板
source：用来存放你的文章
themes：主题
_config.yml: 博客的配置文件
注意：如果hexo init命令没有反应，文件夹下没有node_modules文件夹。此时可退出命令，想办法找个node_modules文件夹复制过来，也是可以的。
（5）此时可以执行hexo g;hexo s命令，打开本地服务器。
4、GitHub创建个人仓库
注意：仓库名字必须是：http://username.github.io，username 是你的用户名。
5、生成SSH添加到GitHub
右键打开git bash，输入命令：
（1）git config –global user.name “yourname”
git config –global user.email “youremail”
（2）验证命令：
git config user.name
git config user.email
（3）生成密钥，复制内容：
ssh-keygen -t rsa -C “youremail”
cat ~/.ssh/id_rsa.pub
（4）打开github，在头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字随便取一个，把id_rsa.pub里面的信息复制进去。
在git bash输入ssh -T git@github.com，如果出现你的用户名，那就成功了。
6、将hexo部署到GitHub
（1）打开博客根目录下的_config.yml文件，修改与博客配置相关的各种信息。
deploy:
type: git
repository: https://github.com/gaoziman/gaoziman.github.io.git
branch: master
（2）安装deploy-git 部署命令：npm install hexo-deployer-git –save
（3）执行命令下面命令就可以看到网站了：
hexo clean
hexo generate
hexo deploy
7、更换主题
（1）进入Hexo官网https://hexo.io/themes/ 选择喜欢的主题
（2）在目录下的themes文件夹下打开git bash，输入命令，下载主题:git clone 复制主题的网址
（3）配置_config.yml文件
8、设置个人域名（不是必要的）
9、发布文章
（1）hexo new “文章 title”
（2）hexo c
hexo g
hexo s
